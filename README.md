### Student Grade Calculator – GUI (Java Swing)

This project is a simple GUI-based Student Grade Calculator built using Java Swing.
The user enters marks for multiple subjects (comma-separated), and the application calculates:

>Total Marks
>Average Percentage
>Grade (A–F)
>
It also includes input validation and error handling.

--------------------------------------------------------------------------------------------------------------------------------

### Features

>User-friendly Graphical Interface using Swing
>Accepts comma-separated marks
>Validates marks (0–100 only)
>Calculates:
    >Total Marks
    >Average
    >Grade
>Displays results instantly
>Error pop-ups for invalid input

-------------------------------------------------------------------------------------------------------------------------------------
### How It Works

 How It Works

1. User enters marks like:
80, 92, 75, 60, 88
2. App splits the text into individual values.
3. Converts each value into an integer.
4. Ensures each mark is between 0 and 100.
5. Adds all marks → Total
6. Computes average → Total / Number of subjects
7. Assigns grade:
 90+  → A
 80+  → B
 70+  → C
 60+  → D
 below 60 → F
8. Displays the results in the GUI.
